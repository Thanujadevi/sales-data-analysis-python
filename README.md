Sales Data Analysis & Reporting Using Python

This project focuses on analyzing structured sales data using Python to extract meaningful business insights. It simulates real-world data analysis tasks performed by junior data analysts and demonstrates how raw sales data can be transformed into useful information for business decision-making.

Project Objective
The main objective of this project is to understand sales performance by calculating total revenue, identifying top and low-performing products, analyzing category-wise sales, and studying monthly sales trends.

Tools Used
Python
Google Colab
Pandas and NumPy
GitHub

Dataset Description
The dataset contains 50 sales records with details such as order ID, order date, product name, category, quantity, price, and total sales amount. The data is structured to resemble real-world retail sales data.

Analysis Performed
Calculation of total sales for each order
Overall revenue and total order count
Product-wise sales performance analysis
Category-wise revenue analysis
Monthly sales trend analysis

Business Insights
High-performing products should be prioritized for inventory restocking.
Low-performing categories may require promotional strategies.
Monthly sales trends indicate seasonal demand patterns.

Output Files
Week2_Sales_Data_Analysis.ipynb
week2_sales_analysis_output.csv

Conclusion
This project demonstrates how Python-based data analysis can support business decision-making by converting raw data into meaningful insights.
